dikarenakan langkanya dataset untuk senjata. melakukan augmeentasi atau dataset sintesis dengan membuat model 3d lalu di render 360 derajat. jika dirender untuk setiap derajat dalam lingkungan bola 3d dengan menempatkan vert sebagai titik kamera dan ini bisa menghasilkan untuk 1 kelas bisa didapati 65,341 gambar. agar mendapatkan banyak gambar lagi dilakukan variasi dengan model bertekstur realis dan hanya base model, serta ditambahkan backgroun misal 10. maka bisa didapati 1.306.820 gambar sintesis untuk 1 kelas.
